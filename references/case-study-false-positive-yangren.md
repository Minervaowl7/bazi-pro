## 误判过程

### 第一步：解读者形成对月令的第一印象

月支**子** = 壬水帝旺 = 羊刃 → 大脑自动锁定"羊刃格"。

### 第二步：第〇步检索被带偏

查询构造为 `"羊刃格 壬水 子月..."` —— 已经预设羊刃格方向。结果全部是羊刃/十恶大败/刑冲相关的条文。

### 第三步：层0筛查"命中即停"（v3.0 缺陷）

层0列表：专旺格 → 从财格 → … → 羊刃格

羊刃格排在前面，而判定条件是极其简单的"月支=羊刃？"→ 抢先从格之前命中。**从格相关条目在层0列表后面，永远没有机会被检查。**

### 第四步：从格三要件排除此命（v3.0 缺陷）

从格三要件只定义了从弱格条件（日主无根 + 无印 + 克泄耗成势）。壬水帝旺+子亥双根 → "日主有根→不回从格，按正格"。

**从强格根本不在 skill 的格局体系里。**

### 结果

```
误判: 羊刃格 → 用神土（官杀制刃）→ 大运戊戌判为吉运
正确: 假从强格 → 用神金水（顺其旺势）→ 大运戊戌判为凶运
喜用神 180° 翻转
```

## 根因分析

| 缺陷 | 表现 |
|------|------|
| 层0缺从强格 | 金水91%的从强格无处归类 |
| 从格三要件只覆盖从弱 | 只定义从弱条件，从强条件完全缺失 |
| 层0排序错误 | 羊刃格只需查月支，极易抢先命中，阻断从格判断 |
| 第〇步无反事实检索 | 检索方向被解读者对月令的第一印象带偏 |

## v3.1 修复措施

1. **层0拆分为A/B**：层0-A 全局格局（先查），层0-B 局部格局（后查）
2. **新增从强格**（真从强/假从强），含三要件判定
3. **羊刃格加前置条件**：仅当从强格/专旺/两行成象均不成立时才按羊刃格论
4. **第〇步双通道**：极偏命盘强制运行顺向+反事实两组检索
5. **新增3条解读红线**：从强格误判、层0顺序、检索带偏
