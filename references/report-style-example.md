# 报告行文风格范例

> 本文件是 bazi-pro v3.4.1 SKILL.md §10.6 的配套参照。展示了"表格+精批并存"的正确写法。
> 关键：结构化数据用表格，分析和故事用叙述；全篇"你"贯穿；表格后有白话翻译。

## 结构比例

| 章节 | 表格/图示 | 叙述解读 |
|------|----------|---------|
| 格局总论 | 从强格三要件表 + 三反馈验证表 | ✅ 开篇大师口吻，用三次验证讲辨伪故事 |
| 盘面精读 | 四柱总览表（天干/地支/藏干/十神/空亡/神煞） | ✅ 逐柱口语化解读，比喻："护身符""壮丁""刀在鞘中" |
| 古籍检索 | 双通道得分对比表 | 简短裁判理由 |
| 五行力量 | 完整计算表 + ASCII柱状图 | ✅ 一句话总结数据含义 |
| 刑冲合害 | 关系速查表 | ✅ 挑最重要的三件事口语化复述 |
| 空亡时间线 | 简表（位置/影响/填实年） | ✅ 关键结论一句话 |
| 大运总览 | 十步大运表 | 一句态势概括 |
| 逐年精批 | 十年概览表（年份/干支/标记/引动/态势） | ✅ 每一年的故事独立展开，大师口吻，有建议 |
| 六维分论 | 性格神煞表、事业行业表、财运节点表、感情窗口表、健康五行表 | ✅ 叙述为主，表格为辅，禁止纯表格 |

## 语气标杆

✅ "你这张命盘，我排出来一看，八个字里头七个字是金水"
✅ "子丑合土——把你月令帝旺之水给合化了！"
✅ "巳申合水这个'老朋友'再次启动——忌神被拉壮丁"
✅ "2024年甲辰——食神出鞘，直斩戊土七杀"

❌ "命主日柱壬申，十恶大败日，财库不聚"（第三人称）
❌ "该命盘具有以下特征：……"（学术论文腔）
❌ 只用表格罗列逐年数据无任何解读
❌ "建议关注""值得留意"等万能句式结尾

## 古籍引用方式

✅ 嵌入叙述：`《滴天髓》曰："假从亦可发其身"——你日支申里那个壬水，就是你的暗生之根。`
❌ 括号堆砌：`（据《滴天髓》通道B）` 孤零零出现在段落末尾
