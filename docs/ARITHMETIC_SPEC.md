本文档映射 bazi-pro 核心计算公式到代码实现，包含输入/输出契约、边界条件、精度规则和古籍依据。

## 1. 五行力量计算 (`calc_element_forces`)

**文件**: `bazi_pro/core/elements.py:194`

### 输入

| 参数 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `bazi_parts` | `list[str]` | 4 个元素，每个 2 字符 | 四柱干支，如 `['甲子', '丙寅', '己卯', '癸酉']` |
| `month_zhi` | `str` | 1 个地支字符 | 月支，如 `'寅'` |

### 输出

```python
{
    'raw': {'木': float, '火': float, '土': float, '金': float, '水': float},
    'percent': {'木': float, ...},          # 原始百分比，总和 ≈ 100.0
    'percent_adjusted': {'木': float, ...}, # 合化修正后百分比，总和 ≈ 100.0
    'total': float,                         # 原始力量总和
    'hehua': dict,                          # 合化检测详情
}
```

### 计算规则

#### 1.1 天干力量

| 条件 | 力量值 | 古籍依据 |
|------|--------|---------|
| 天干有根（地支藏干含同五行本气/中气） | 1.0 | 《子平真诠》"干多不如根重" |
| 天干无根（虚浮） | 0.4 | 同上 |

#### 1.2 地支藏干力量

| 气 level | 权重 | 古籍依据 |
|----------|------|---------|
| 本气 | 1.0 | 《三命通会》藏干体系 |
| 中气 | 0.6 | 同上 |
| 余气 | 0.3 | 同上 |

**月令加成**: 月支本气权重 × 1.5（用 `zhi == month_zhi` 判断，非硬编码索引）

#### 1.3 合化力量转移

| 合化类型 | 转移比例 | 古籍依据 |
|----------|---------|---------|
| 天干合化（紧贴） | 50% | 《子平真诠》"合而化者，两失其本气" |
| 天干合化（遥合） | 0%（不转移） | 遥合力弱 |
| 三合局 | 50% | 《三命通会》"三合者，生旺墓之合" |
| 半合局 | 20% | 《三命通会》"若三字缺一则化不成局" |
| 会方 | 30% | 《滴天髓》"方是方兮局是局" |
| 六合 | 25% | 《神峰通考·安星辰法》 |

#### 1.4 边界条件

| 条件 | 处理 | 代码位置 |
|------|------|---------|
| 力量总和为 0 | `max(0.01, sum(...))` 防止除零 | `elements.py:356` |
| 合化修正后力量为负 | `max(0, ...)` 截断为 0 | `elements.py:362` |
| 天干合化争合 | `contested = True`，标记削弱 | `elements.py:145` |

#### 1.5 精度规则

- 百分比保留 1 位小数: `round(v / total * 100, 1)`
- 力量值保留 2 位小数: `round(total_raw, 2)`

## 6. 缓存键计算

**文件**: `server/analysis.py`

### 缓存键格式

```
bazi:v5:{sha256(payload_json)[:24]}
```

### 缓存键包含字段

- 八字、日主、性别、detail_level、school
- 大运列表（如有）
- 阳历/农历（如有）

- 默认: 3600 秒（1 小时）
- 可配置: `BAZI_CACHE_TTL` 环境变量
